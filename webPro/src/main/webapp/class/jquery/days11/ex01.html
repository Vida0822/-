1. jQuery는 무엇입니까?
	1) jQuery는 빠르고, 작고, 기능이 풍부한 JavaScript 라이브러리입니다.
	2) 다양한 브라우저에서 작동하는 사용하기 쉬운 API를 사용하여 
	 HTML 문서 탐색 및 조작, 이벤트 처리, 애니메이션 및 Ajax와 같은 작업을 **훨씬 간단**하게 만듭니다.
