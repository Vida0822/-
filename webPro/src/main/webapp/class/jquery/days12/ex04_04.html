jquery method : 애니메이션 효과 정지(중지)

$("선택자").stop(stopAll, goToEnd); 
:현재 애니메이션 효과가 완료되기 전 중지할 때 사용하는 메서드 

	A 	->	B -> 	C [대기열] 
			stop() 
	
1) stopAll ->  true, false 
B에 stop() 을 주면 B는 멈추는데, 남아있는 C는 어떻게할거니? 
-> true: 멈출래~
-> false (default): 안멈출래 

2) goToEnd 
: 현재 진행중인 애니메이션을 즉시 완료할 수 있는 여부를 지정하는 매개변수 
true, false(default)
