7. 쿠키형식 
	이거 _언더바가 아니라 걍 공백표시한거임 / [] : 옵션 
	
	쿠키이름 = 쿠키값;_ [만료시점=날짜;_경로=경로값;_도메인=도메인명;_보안처리옵션] 
	name = 쿠키값;_expires=날짜;_path=경로값;_domain=도메인명;_secure
	
	
	* 만료시점을 정하지 않으면 기본적으로 쿠키는 브라우저를 닫으면 삭제된다 
	* 경로를 생략하면 기본적으로 쿠키는 현재 페이지에 속하는 쿠키
