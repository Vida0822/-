background-size 속성 : contain, cover

- contain : 배경 이미지를 최대한 크게 조정한다
            (단, 너비와 높이가 모두 콘텐츠 영역에 맞아야 한다)
            ( 긴 길이 기준으로 화면 안에 알맞게 들어감)
- cover : 콘텐츠 영역이 완전히 배경 이미지에 포함되도록 처리한다
          배경 이미지의 일부는 배경 위치 지정 영역에서 보이지 않을 수도 있다
          ( 짧은 길이 기준으로 화면에 꽉 채움 )
