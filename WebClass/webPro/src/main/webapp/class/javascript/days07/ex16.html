[js Object(객체, 개체)]
1. js의 객체는 기본형을 제외한 모두가 js 객체이다! 
2. js 기본형 ? string , number,  boolean, null, undefined, bigInt 
3. js 객체 ? object , array , function / new String() , new Number() , new Date(), new Boolean / Math 등등 
4. js 배열 []
5. js 객체 {} -> js 객체도 변수로 받을 수 있다.  
ex) let person = {name : "홍길동", age : 20} ; 
	ㄴ name, age -> 객체의 속성 / info : function(){} -> 객체의 메서드   
	
	/*위랑 같은 코딩 */ 
const person = {};  // 객체만 먼저 생성 
// (같음) const person = new Object(); 
 person.name = "홍길동"; // 생성 후 속성 추가 
 person.age = 20; 

// person 객체가 이미 존재할 때 새로운 속성을 지정하기
