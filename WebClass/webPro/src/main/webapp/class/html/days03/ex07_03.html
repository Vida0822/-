1. html 모든 태그(요소)에는 유형에 따라 기본 표시(display)값이 있다
 2. 표시(display)
 	ㄱ. 블럭모드 : 전체 너비
 				   p, hn, dl, dt, dd, ol, ul, li, hr, blockquote, [div] 등등
 	ㄴ. 인라인모드 : 새 라인(줄)에서 시작 X, 필요한 너비만큼만 차지한다
 					 img, a, span, i, q, sub, sup, b, button, input 등등
 3. div 요소
 	1) 웹 페이지의 배치( layout )
 	2) 블럭 모드
 	3) 다른 html 요소의 컨테이너 역할
