3. jquery, [css]의 selector(선택자) ex02.html
	1) 태그명, #ID, .클래스명
	   ID명은 숫자로 시작 X
	2) 요소들 간의 특정 관계 ex03.html
	3) 의사 클래스( 특정 상태 기반 요소 )
	   :의사 클래스명
	4) 의사 요소( 요소의 일부 선택 및 스타일 지정 )
	   ::의사 요소
	   #section::after {
	   	clear: both;
	   	display: table;
	   	content: "";
	   }
	5) 속성

	   [속성명$=속성값]
