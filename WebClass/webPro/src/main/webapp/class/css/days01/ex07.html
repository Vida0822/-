	여백( margin )
	테두리
	패딩( padding )
	
	margin-top, right, bottom, left
	margin 속기(약식)
	
	여백( margin ) 속성값
	1) 길이 단위 : px, pt, cm 등등
	2) %( 기준 부모요소의 % )
	3) auto - 브라우저가 여백을 자동으로 설정
	4) inherit(상속) - 부모 여백 -> 자식 여백 설정
