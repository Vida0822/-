css display 속성

[display 속성]
1. 레이아웃을 제어하기 위한 가장 중요한 css 속성
2. 기본값 : block, inline
   block - div, p, header, hn, hr 등등
     ㄴ 특징 : 항상 새로운 라인에서 시작
               부모 너비의 사용 가능한 전체 너비( width 100% )
   inline - a, img, span 등등
     ㄴ 특징 : 새로운 라인에서 시작 X
               너비는 필요한 내용(content)만큼만 차지한다
3. 요소가 표시되는 여부를 지정하는 속성
4. display: none;
   요소를 삭제 또는 숨기기 + js
5. li 블럭모드
   [       100%       ] - block
   li {
       display: inline;
   }
