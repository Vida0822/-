[overflow 속성]
- 요소의 내용이 너무 커서 지정된 영역에 맞지 않을 때
  그 내용을 자르거나, 스크롤바를 추가해서 지정
- overflow:
	ㄴ visible : 내용이 커서 넘치더라도 잘리지 않는다( 기본 )
	ㄴ hidden : 내용이 커서 넘치면 잘라낸다
	ㄴ scroll : 무조건 스크롤바가 추가된다
	ㄴ auto : 자동( 내용이 커서 넘치면 스크롤바가 추가된다 )
- 참고) overflow 속성은 지정된 높이의 블럭 요소에만 적용된다
