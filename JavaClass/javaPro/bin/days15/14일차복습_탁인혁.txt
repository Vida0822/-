5. NullPointerException 예외(에러)는 언제 발생하는 가 ? 
	변수가 null인 상태에서 객체의 데이터나 메소드를 사용하려 할 때 발생한다
6. 클래스의 선언 형식
	[접근지정자] [기타제어자] class 클래스명 [extends 슈퍼클래스] [implements 인터페이스...] {}
7. 클래스 앞의 접근지정자의 종류에 대해서 설명하세요.
	public	 : 패키지 내부/외부 어디서든 상속 및 참조 가능
	default	 : 패키지 내부에서만 상속 및 참조 가능 (==package)
	private	 : 독립 클래스 X, 중첩 클래스 O / 같은 파일 내에서만 상속 및 참조 가능
	protected : 독립 클래스 X, 중첩 클래스 O / 패키지 내부에서만 상속 및 참조 가능, 패키지 외부에서는 상속만 가능
8. 멤버 앞의 접근지정자의 종류에 대해서 설명하세요.
	public	 : 패키지 내부/외부 어디서든 접근 및 참조 가능
	default	 : 패키지 내부에서만 접근 및 참조 가능 (==package)
	private	 : 클래스 내에서만 접근 및 참조 가능
	protected : 패키지 내부에서만 접근 및 참조 가능
9. 클래스 선언할 때 기타제어자 종류에 대해서 설명하세요.
	abstract	: 추상 클래스
	final	: 최종 클래스
	static	: 정적 클래스 / 독립 클래스 X, 중첩 클래스 O
10. 클래스 복사 ( 객체 복사 )에 대해서 설명하세요. ( 수업 예제 )
	메서드의 매개변수를 클래스 참조타입으로 해서 그 클래스에 다른 클래스를 넣어 복사하는 것을 클래스 복사라고 한다
11. 클래스 배열( 객체 배열 ) 에 대해서 설명하세요. ( 수업 예제 ) 
	배열 선언을 할 때 자료형의 위치에 클래스를 적용시킨 것을 클래스 배열이라고 한다
12.  선언되는 위치를 기준으로 변수 구분하고, 생성시기, 소멸시기에 대해서 설명하세요.
